📐 KEY EQUATIONS BY CATEGORY
================================================================================

LINEAR REGRESSION EQUATIONS:
──────────────────────────
1. Model:  y = wx + b

2. Energy:  E(w,b) = Σᵢ(yᵢ - (wxᵢ + b))²

3. Solution:  w* = (X^T X)^(-1) X^T y

4. Bias:  b* = ȳ - w*x̄

NONLINEAR BASIS FUNCTIONS:
─────────────────────────
1. General Form:  y = Σₖ wₖ bₖ(x)

2. Polynomial Basis:  bₖ(x) = x^k

3. RBF Basis:  bₖ(x) = exp(-(x-cₖ)²/(2σₖ²))

4. Regularized Objective:  E(w) = ||y - Bw||² + λ||w||²

ARTIFICIAL NEURAL NETWORKS:
──────────────────────────
1. Sigmoid Activation:  g(a) = 1/(1 + e^(-a))

2. Single Hidden Layer:  y = Σⱼ wⱼ⁽²⁾ g(wⱼ⁽¹⁾x + bⱼ) + b

3. Regularized Loss:  E(w,b) = ||y - f(x)||² + λ||w||²

PROBABILITY THEORY:
──────────────────
1. Product Rule:  P(A,B) = P(A|B)P(B)

2. Sum Rule:  P(A) + P(¬A) = 1

3. Bayes Rule:  P(A|B) = P(B|A)P(A)/P(B)

4. Marginalization:  P(B) = ∫P(A,B)dA

5. Independence:  P(A,B) = P(A)P(B) if independent

GAUSSIAN DISTRIBUTION:
─────────────────────
1. Univariate:  G(x|μ,σ²) = (1/√(2πσ²)) exp(-(x-μ)²/(2σ²))

2. Multivariate:  G(x|μ,Σ) = 1/(√((2π)^D|Σ|)) exp(-½(x-μ)^T Σ^(-1)(x-μ))

3. Standard Form:  (x-μ)^T Σ^(-1)(x-μ) = Σᵢ zᵢ² (after diagonalization)

MAXIMUM LIKELIHOOD ESTIMATION:
──────────────────────────────
1. ML Principle:  θ_ML = argmax P(D|θ)

2. MAP (with prior):  θ_MAP = argmax P(D|θ)P(θ)

3. Gaussian Parameters:  
   μ_ML = (1/N)Σxᵢ
   Σ_ML = (1/N)Σ(xᵢ - μ)(xᵢ - μ)^T

LOGISTIC REGRESSION:
──────────────────
1. Model:  P(C₁|x) = 1/(1 + exp(-(w^T x + b)))

2. Decision Boundary:  w^T x + b = 0

3. Loss (Bernoulli):  E(w) = -Σᵢ[yᵢ ln p(yᵢ|xᵢ) + (1-yᵢ)ln(1-p(yᵢ|xᵢ))]

K-NEAREST NEIGHBORS:
───────────────────
1. Unweighted:  y_new = (1/K) Σᵢ∈NK yᵢ

2. Weighted:  y_new = (Σᵢ∈NK w(xᵢ)yᵢ) / (Σᵢ∈NK w(xᵢ))

3. Weight Function:  w(xᵢ) = exp(-||x - xᵢ||²/(2σ²))

GRADIENT DESCENT:
────────────────
1. Update Rule:  w_{t+1} = w_t - λ∇E(w_t)

2. Gradient:  ∇E = [∂E/∂w₁, ..., ∂E/∂wₙ]^T

3. Line Search:  λ* = argmin E(w_t - λ∇E(w_t))

PRINCIPAL COMPONENT ANALYSIS:
─────────────────────────────
1. Model:  y = Wx + b

2. Covariance:  K = (1/N)Σ(yᵢ - ȳ)(yᵢ - ȳ)^T

3. Eigenvector Solution:  VAV^T = K

4. Transform:  x = W^T(y - b)

5. Variance Captured:  var(xⱼ) = λⱼ

K-MEANS CLUSTERING:
───────────────────
1. Objective:  E = ΣᵢΣₖ zᵢₖ ||xᵢ - μₖ||²

2. Update Centers:  μₖ = (Σᵢzᵢₖxᵢ)/(ΣᵢzᵢₖK)

3. Assignment:  zᵢₖ = 1 if k = argmin ||xᵢ - μₖ|| else 0

BAYES RULE (Inference):
──────────────────────
1. Posterior:  P(θ|D) = P(D|θ)P(θ)/P(D)

2. Likelihood:  P(D|θ) = ∏ᵢ P(dᵢ|θ)

3. Normalization:  P(D) = ∫P(D|θ)P(θ)dθ

BAYESIAN REGRESSION:
───────────────────
1. Prior:  w ~ N(0, α^(-1)I)

2. Likelihood:  p(y|X,w) = N(Bw, σ²I)

3. Posterior:  P(w|X,y) = N(w̄, K)
   where: K = (B^T B/σ² + αI)^(-1)
          w̄ = K B^T y/σ²

4. Prediction:  P(y_new|D,x_new) = N(μ_pred, σ²_pred)

MARKOV CHAIN MONTE CARLO:
─────────────────────────
1. Transition:  x_{t+1} ~ q(x'|x_t)

2. Acceptance Ratio:  α = min(1, P*(x')q(x_t|x')/P*(x_t)q(x'|x_t))

3. Metropolis Update:  Draw u ~ Uniform(0,1)
                       if u < α: x_{t+1} = x'
                       else: x_{t+1} = x_t

LAGRANGE MULTIPLIERS:
────────────────────
1. Lagrangian:  L(x,λ) = E(x) + λg(x)

2. KKT Condition:  ∇E + λ∇g = 0

3. Constraint:  g(x) = 0

🔑 FUNDAMENTAL CONCEPTS
================================================================================

LEARNING PARADIGMS:
──────────────────
1. Supervised Learning: Learn mapping from labeled data
2. Unsupervised Learning: Discover structure in unlabeled data
3. Reinforcement Learning: Learn policies from rewards

MODEL COMPLEXITY:
─────────────────
• Overfitting: Model fits training noise, poor generalization
• Underfitting: Model too simple, cannot capture patterns
• Regularization: Add penalty term to control complexity
• Cross-Validation: Assess generalization on held-out data

OPTIMIZATION APPROACHES:
────────────────────────
• Closed-form Solutions: Direct algebraic solution (e.g., Linear Regression)
• Iterative Optimization: Gradient-based methods (e.g., Gradient Descent)
• Expectation-Maximization: For models with latent variables

PROBABILISTIC INFERENCE:
────────────────────────
• Maximum Likelihood: Choose θ that maximizes P(D|θ)
• MAP Estimation: Incorporate prior P(θ)
• Bayesian: Maintain full posterior P(θ|D)
• Sampling: Use Monte Carlo when inference is intractable
